Basic_git_commands_for_uploading

2020-08-11

After initial configuration of your github or gitlab credentials (see <href="https://education.github.com/git-cheat-sheet-education.pdf">here), there are very few commands you need to worry about to upload your content to the server:

Add changes to git.

git add .

Commit changes.

git commit -m “your message for the log”

and finally

Push source and build repos.

git push origin master
